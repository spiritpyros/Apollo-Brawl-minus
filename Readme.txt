The forwarder can be used for the USB loader and the Disc launcher:
USB loader users please delete the "Smash_CE" folder inside the "apps" folder in order to use the forwarder channel provided inside the "wad" folder. 
Vice Versa, Disc users please delete the "smash_ce_usb_loader" folder inside the "apps" folder in order to use the forwarder.

In order to update, please launch the "Super Smash Bros. CE Boot Menu.exe" provided and check the newsfeed on top! If an update is available,
simply hit the "update" button and everything will be updated automatically. After it is done updating, you can eject your sd card and play!
